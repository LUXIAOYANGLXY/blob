题目一
解答：
他有提高获奖的可能性
提高1/3
分析如下：
选手在不改变选择时拿到奖品的概率为P(A)=1/3,也就是选手选择的那扇门正好是有奖品的那扇门，概率为1/3；
选手在改变选择时拿到奖品的概率为P(B)=2/3,因为主办方已经打开了一扇没有奖品的门，此时选手拿到奖品的条件变为了第一四选手选择的那扇门是没有奖品的，概率为2/3
